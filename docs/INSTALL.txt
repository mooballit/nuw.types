nuw.types Installation
----------------------

To install nuw.types using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``nuw.types`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        nuw.types
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
